## 문제

오라클에서 다른 곳에서 제공받은 뷰테이블을 START WITH 로 계층 조회하는 쿼리 실행 중

> "원격 데이터베이스에서는 start with 하위 질의를 사용할 수 없습니다." 오류 발생.

## 원인

DBLINK를 걸어 사용 중인 테이블 또는 뷰 등에서는 해당 구문을 사용할 수 없음.

## 해결

해당 뷰 테이블을 우리 쪽 스키마로 동일하게 테이블 및 데이터를 복사 하여 사용.
